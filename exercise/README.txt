
Exercise 1:
It is done using Python 2.7. To run it, type the command "python exercise1.py" in the terminal. To test the program, you can either change the value of "array" and "n" in the main function, or you can add test case in the unit test class.


Exercise 2:
It is done using C. To run it, first, type the command "gcc exercise2.c" in the terminal, and then, type the command "./a.out".
To test the program, you can change the value of "cells[]" for initial configuration, and "k" for the number of steps in the main function.
